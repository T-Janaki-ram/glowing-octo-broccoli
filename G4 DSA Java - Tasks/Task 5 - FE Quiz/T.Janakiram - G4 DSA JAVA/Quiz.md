1. Which HTML5 element is most appropriate for representing self-contained content that could be independently distributed, such as a blog post?

A. `<section>`

B. `<article>`

C. `<aside>`

D. `<main>`

Answer: B) `<article>`

2. Which attribute is used to provide alternative text for an image when the image cannot be displayed?

A. title

B. alt

C. caption

D. description

Answer: B) alt

3. Which HTML5 input type is designed to accept an email address and can trigger email-specific validation in browsers?

A. text

B. mail

C. email

D. url

Answer: C) email

4. Which element is used to embed playable video content directly in a web page?

A. `<media>`

B. `<video>`

C. `<movie>`

D. `<source>`

Answer: B) `<video>`

5. What is the purpose of the `<label>` element in a form?

A. To create a visible heading for the page

B. To associate descriptive text with a form control

C. To validate form data automatically

D. To submit a form without JavaScript

Answer: B) To associate descriptive text with a form control

6. Which attribute makes a form field mandatory before submission?

A. mandatory

B. required

C. validate

D. checked

Answer: B) required

7. Which element is best suited for site navigation links?

A. `<nav>`

B. `<menuitem>`

C. `<links>`

D. `<aside>`

Answer: A) `<nav>`

9. Which element represents the main unique content of a page?

A. `<body>`
